## Выводы

### Победитель: Test-Fixed-30

- **Лучшая точность**: 40% (4/10 правильных ответов в топ-1)
- **Лучшее среднее расстояние**: 0.636 (меньше = точнее)
- **Больше чанков**: 302 чанка обеспечивают лучшее покрытие

### Рекомендации

1. **Для RAG используй FixedChunker с chunk_size=30-50**
   - Маленькие чанки дают лучшую точность
   - Больше перекрытие (5-10) улучшает релевантность

2. **StructureChunker показал себя хуже**
   - 20-30% точность vs 40% у fixed
   - Возможно, из-за отсутствия четкой структуры в файлах AI Challenge

3. **Размер имеет значение**
   - 30 слов: 40% точность
   - 50 слов: 30% точность  
   - 100 слов: 30% точность
   - Большие чанки = больше "шума"

4. **Контекст для LLM**
   - При маленьких чанках может потребоваться увеличить top_k до 5-10
   - LLM получит больше релевантных фрагментов
